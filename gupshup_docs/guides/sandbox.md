# Sandbox

Sandbox is a dedicated testing environment that allows businesses to simulate and validate their WhatsApp Business API integration before going live. This secure space enables experimentation with messaging features, configuration settings, and overall functionality without affecting real users or production data.

In the sandbox environment, businesses can:

* Test access API functionalities, with applicable charges deducted from the **[Wallet](https://docs.gupshup.io/docs/wallet)** balance.
* Send messages to desired recipients using our sandbox number + 91 70XXXXX90.
* Customize recipient numbers by specifying the country code and WhatsApp number.
* Enter and send custom text messages or attach sample media.
* Utilize predefined sample interactions for interactive messages.
  > 📘
  >
  > Button-type templates with **\[ ]** indicate static quick reply buttons. These are non editable and if edited may result in send message failure on callback
* Receive inbound messages from customers, with the list updated as messages are received.
