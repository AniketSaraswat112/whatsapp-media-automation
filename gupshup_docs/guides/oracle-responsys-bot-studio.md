# CX- Oracle Responsys : Gupshup Bot Studio for Oracle Responsys

Start Automated WhatsApp Campaigns and engage your customer from Responsys

# Overview

The **Gupshup Bot Studio for Oracle Responsys** allows you to integrate your Gupshup Console WhatsApp Business Account with Oracle Responsys. This integration helps you engage with your customers seamlessly on WhatsApp by sending personalized, automated campaigns using text and multimedia messages.

With this integration, you can leverage Oracle Responsys Program Designer to build automated, multi-channel campaigns, which can be executed via WhatsApp. It allows for two-way interactions, ensuring better customer engagement. You can also personalize messages, streamline communications, and optimize WhatsApp messaging to improve customer satisfaction.

<br />

# Features

* **Seamless Integration**: Directly integrate Gupshup WhatsApp account with Oracle Responsys Program Designer, eliminating the need for additional tools.
* **Automated Interaction**: Implement a chatbot to handle common customer queries, provide product recommendations, or guide users through the sales funnel. The bot can use Program Designer to create automated responses based on predefined triggers.
* **Intelligent Routing**: Design the bot flow to handle specific customer inquiries, like order status or promotional offers, and provide relevant responses instantly.
* **Instant Responses**: Set up automatic replies for frequently asked questions (FAQs). For example, a customer asks about your business hours or return policies, and the bot instantly provides the correct information.

<br />

# Prerequisites

Before you can integrate the Gupshup Bot Studio with Oracle Responsys, the following prerequisites are required:

* **Gupshup Console Account**: You must have an active Gupshup Console account.
* **Valid User ID and Project ID**: Ensure that you have a valid User ID and Project ID within the Gupshup Console. This is needed for integration purposes.
* **Project Compatibility with Oracle Responsys**: The project in your Gupshup Console must be compatible with Oracle Responsys. Compatibility will be verified during the Gupshup onboarding process by the onboarding team.
* **Oracle Responsys Account**: You need an active Oracle Responsys account to create and manage your campaigns. The Gupshup integration will work seamlessly once the Oracle Responsys account is set up.
* **Webhook Configuration**: Webhooks need to be configured to handle delivery reports and incoming messages from WhatsApp, as well as to ensure smooth data transfer between Gupshup and Oracle Responsys.
