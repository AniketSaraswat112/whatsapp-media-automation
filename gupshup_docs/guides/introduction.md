# Introduction

* To send a WhatsApp message from Dynamics 365 CRM to your client using Gupshup you need to install the solution  from the Microsoft App Source using this [link](https://appsource.microsoft.com/en-us/product/web-apps/gupshuptechnologyindiaprivatelimited1717507324467.gupshup?ocid=GTMRewards_WhatsNewBlog_gupshup_08292024).
