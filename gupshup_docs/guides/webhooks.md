# Webhooks

Webhooks are a mechanism used to receive real-time notifications about specific events occurring within the account. They are HTTP callbacks that allow a server to communicate with your application when certain events take place, ensuring that your system can react to these events promptly and efficiently.

  Webhooks Tab

For more detailed information about webhooks and how they can be used within your WhatsApp Business Account, please refer to this resource: [Understanding Webhook Key Points](https://docs.gupshup.io/docs/what-is-a-webhook) & [How to set Callback URL](https://docs.gupshup.io/docs/set-webhookcallback-url)

You have the option to select checkboxes for both **Message events** and **System events** to ensure you receive notifications on your webhook.

> 📘
>
> By default, **Enqueued** and **Failed** message events will be turned on, ensuring that you receive notifications for these specific events.
