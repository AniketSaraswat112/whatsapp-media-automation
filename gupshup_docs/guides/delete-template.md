# Delete Template

## Deleting A Template

1. Go to Dashboard > WhatsApp > Your App > Templates, and click the **Arrow** button next to **Manage** on an existing template.

   <Image alt="Edit Template" align="center" border={true} src="https://files.readme.io/067707f-Screenshot_2024-05-29_at_4.09.07_PM.png">
     Delete Template
   </Image>
2. Select **Delete**. A window will pop up to confirm the deletion.

   <Image alt="Confirm Deletion" align="center" border={true} src="https://files.readme.io/5bf3ea2-Screenshot_2024-05-29_at_10.39.28_PM.png">
     Confirm Deletion
   </Image>
3. Select **Delete**. Your template has been successfully deleted.
