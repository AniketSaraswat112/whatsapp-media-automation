# Settings

The **Settings** tab in the App dashboard is where you can customize various preferences and configurations related to your app experience.

<Image alt="Settings Tab" align="center" border={true} src="https://files.readme.io/135443f-Screenshot_2024-05-30_at_2.07.04_PM.png">
  Settings Tab
</Image>

| Key             | Description                                     |
| --------------- | ----------------------------------------------- |
| App name        | Name of the app                                 |
| Hosted In       | Region of the hosted App (e.g., India)          |
| App ID          | Unique identifier for the application           |
| App Type        | Type of the application (e.g., Live or Sandbox) |
| WABA ID         | WhatsApp Business Account ID                    |
| Messaging Limit | Limitations on messaging usage                  |
| App API Key     | API key for accessing the app with API          |

> 🚧
>
> You might notice a **Delete App** button next to the App name. Once your app is live, the "Delete App" button will be disabled.
