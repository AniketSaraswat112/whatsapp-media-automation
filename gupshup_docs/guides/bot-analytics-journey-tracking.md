# Journey Tracking

### Journey Tracking

A Journey is a path that defines a purpose for which the user has initiated a conversation. It is a logical sequencing of nodes that defines a Journey of events.

This metric shows the top 5 Journeys for which maximum users have been observed. This graph helps you analyze the day by day trend of the Journeys which are most commonly used by the customers.

By default the data showcases the top 5 Journeys from all the channels, for the last 7 days; filters present above the graph can be used to track any Journey for a duration of your choice.

### Node Tracking

This graph is dependent on the Journey tracking graph, it provides the top nodes among the top Journeys. A user can change the view by either selecting the node from the parent chart or from the dropdown.

Upon selecting the checkbox 'View Nodes Independently' present above the graph and deselected by default, a user will be able to select and view the trend graph against any node which has been selected from the drop down present.

By default the data showcases the top 10 nodes from the top 5 Journeys across all channels, for the last 7 days.
