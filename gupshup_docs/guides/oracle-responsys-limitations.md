# Limitations

1. Supported multimedia messages, formats, and maximum file size :
   1. **Image** :
      1. Supported Format : **png** and **jpeg**
      2. Max file size : 4 MB
   2. **Video** :
      1. Supported Format: **mp4**
      2. Max file size: 4 MB
   3. **Document** :
      1. Supported Format : **pdf**
      2. Max file size : 4 MB
2. Multimedia templates only support public URLs
