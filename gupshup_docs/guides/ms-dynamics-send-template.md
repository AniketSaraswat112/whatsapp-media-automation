# Send Template Messages

Template messages are Highly Structured(HSM)/ Notification messages. Once your WhatsApp Business API is Live, you can create template messages and submit them to WhatsApp for approval. You can send Template

messages to users that you have opted-in. To know how you can opt-in users, read frequently asked questions.\
Once the app has been installed, the user starts sending WhatsApp messages.

To send WhatsApp messages, click on Contacts/Lead in the left navigation. After clicking, it will redirect to the contact list page.

• Click on a lead from the list, and the system will redirect to the contact details of the selected lead.\
• Once redirected to the lead detail page, look for the “Send Message” button on the ribbon command.
