# Template Messaging

The Template Messaging feature in Gupshup's WhatsApp Message Integration allows you to easily configure and send WhatsApp template messages. This documentation will guide you through the steps to set up template messaging using the Gupshup platform.

Before you start sending WhatsApp Template Messages, make sure you have personalized your template messages and approved to include relevant recipient information.

## Accessing Template Messaging

* Log in to your Zoho CRM system with the appropriate credentials.
* In the dashboard or navigation menu, locate and click on the **Lead** Module.
* Inside the Lead Module, locate the lead record you want to communicate with.Click on the lead record to open and view its details.
* Within the lead record view, look for the **Send WhatsApp Message** Button.
* Click on the WhatsApp Message Button to initiate a conversation or send a WhatsApp message to the associated contact.
* A popup window will appear with three tabs: Template Messaging, session messaging and Channel Configuration.
* For template messaging, focus on the Template Messaging tab.

## Choose Channel

* Inside the Template Messaging tab, you'll find a dropdown menu labeled **Choose Channel**. This dropdown allows you to select the messaging channel you want to use for sending the WhatsApp template message.\
  **Note**: the available channels are pre-configured in the **Channel Configuration** screen. Select the appropriate channel from the dropdown menu.

<Image alt="choose channel" align="center" border={true} src="https://files.readme.io/34bb817-Choose_channel.png">
  choose channel
</Image>

## Populate Approved Templates Table:

* Based on the channel you selected, a table will be populated with all the approved WhatsApp message templates associated with that channel. These templates adhere to WhatsApp's guidelines and policies. Carefully review the available templates in the table and select the one that best suits your messaging needs for this particular step in your journey.
  * Template Name: This column contains the names of approved WhatsApp message templates.
  * Template Body: Here, you can see a preview of the template message content.
  * Select Button: Each template will have a Select button associated with it.
* You have the option to filter templates based on their category, which can include text, image, video and document. To filter templates by category, use the dropdown menu labeled **Filter by Category**.\
  You also have the option to search for specific templates using the **Search Template** feature.\
  To choose a template for your message, click the **Select** button associated with the template that best suits your messaging needs for this particular step in your journey.
