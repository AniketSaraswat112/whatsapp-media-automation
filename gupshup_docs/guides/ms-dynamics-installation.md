# Installation

To install the Gupshup solution please follow the below steps.

**Step 1: Import**

* Click on the “**Import solution**” from the solution navigation.

<Image align="center" className="border" border={true} src="https://files.readme.io/7df5e2c-2024-06-05_15-10.png" />

click on the browse choose

<Image align="center" className="border" border={true} src="https://files.readme.io/212a87e-2024-06-05_15-12.png" />

**Step 2: Import**

* Once you click on the button select the solution provided to you and click ok

<Image align="center" className="border" border={true} src="https://files.readme.io/7afc42e-2024-06-05_15-14.png" />

**Step 3: Version**

* You will see the solution version and solution details click next.

<Image align="center" className="border" border={true} src="https://files.readme.io/73e9d8a-image.png" />

**Step 4: set orgId**

* Replace the default OrgId with the copied OrgId in an earlier step, please do not keep the default orgId it may not work as expected.

  <Image align="center" className="border" border={true} src="https://files.readme.io/cfb9cf0-2024-06-05_15-16.png" />

**Step 5: Import Progress**

* Once the import process starts you will be able to see the progress at the top.

<Image align="center" className="border" border={true} src="https://files.readme.io/64864c9-image.png" />

**Step 6: Import successfully.**

* Once the import process is completed you will be able to see the message “Solution ‘**Gupshup\_mainSolution’ imported successfully**”.

  <Image align="center" className="border" border={true} src="https://files.readme.io/a49392c-image.png" />

### **Uninstall:**

* If you want to remove the solution from your environment, you can simply remove it by following the steps below.\
  Step 1: Select Solution\
  You need to select Gupshup\_mainSolution from the installed environment and select delete as shown below.

<Image align="center" className="border" border={true} src="https://files.readme.io/05cd934-image.png" />

**Step 2: Delete Solution**

* Once you click on delete you will get a popup to confirm the last time before deleting, as once you click on Delete this cannot be rolled back.

  <Image align="center" className="border" border={true} src="https://files.readme.io/31928d8-image.png" />

### **Step 3: Progress**

* Once you confirm you will have the below progress message.

<Image align="center" className="border" border={true} src="https://files.readme.io/077c709-image.png" />

Once it is deleted you will be notified with the message “**Successfully deleted solution**”

<Image align="center" className="border" border={true} src="https://files.readme.io/015f3fe-image.png" />
