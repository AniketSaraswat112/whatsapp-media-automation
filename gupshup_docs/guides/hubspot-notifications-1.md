# Receive Notifications for Incoming Messages

Notifications are alerts or messages that inform users about important events, updates, or actions, such as incoming messages from contacts.

### Overview

This feature allows hubspot user to show notification when a new message is received from contacts. In order to make use of this feature, the existing hubspot Users need to follow the steps mentioned below :

<Image align="center" className="border" border={true} src="https://files.readme.io/7e791e8-2024-05-24_20-47.png" />

* Click the notification button to view which contact has sent the message.
* Click on WhatsApp Reply button to see the new message receive.
