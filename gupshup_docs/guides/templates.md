# Templates

The **Templates** tab is a section within the app dashboard where you can create and manage message templates. You can create new templates, view sample templates, view template status, manage existing templates, compare different templates, etc.

This tab is typically used for organizing and maintaining standardized message formats that can be reused for various purposes, such as customer communications, notifications, or marketing campaigns.
