# Send Template Messages

**Template messages** are Highly Structured(HSM)/ Notification messages. Once your WhatsApp Business API is Live, you can create template messages and submit them to WhatsApp for approval. You can send Template messages to users that you have opted-in. To know how you can opt-in users, read [frequently asked questions](https://support.gupshup.io/hc/en-us/articles/360012075919).

Once the HubSpot app has been installed in the account, user start sending WhatsApp messages.

* To send WhatsApp messages, click on Contacts at the top of the navigation bar. After clicking, it will redirect to the contact list page.

<Image alt="select contacts" align="center" border={true} src="https://files.readme.io/16f9d8a-image_1.png">
  select contacts
</Image>

* Click on a contact from the list, and the system will redirect to the contact details of the selected contact.
* Once redirected to the contact detail page, look for the Gupshup CRM card on the right side of the page.

<Image align="center" className="border" border={true} src="https://files.readme.io/7ee4b49-image_3.png" />

* Click on the card button named "Send WhatsApp Message" on the Gupshup CRM card.\
  Once the button "Send WhatsApp Message" is clicked, a pop-up screen will open

# choose channel

* Inside the Template Messaging tab, you'll find a dropdown menu labeled **Choose Channel**. This dropdown allows you to select the messaging channel you want to use for sending the WhatsApp template message.\
  **Note**: the available channels are pre-configured in the **Channel Configuration** screen. Select the appropriate channel from the dropdown menu.
