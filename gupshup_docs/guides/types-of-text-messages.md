# Types of text messages

You can send following types of text messages

**English**

You can type up to 8000 characters if your text message is in English.

![](https://files.readme.io/2dd4f11-image.png)

**Other Languages**

You can type up to 2000 characters, if your text message is in another language.

![](https://files.readme.io/627fbc6-image.png)

* The ability to read text messages in other languages depends on the receiver’s handset model (compatibility) and the mobile operators.
* One uni-code character is equivalent to four English characters for text message sent in a language other than English (Unicode message).
