# Copy Journey

### Introduction:

Now you can make a copy of your journeys with just a click of a button. You can customize the copied journey and use it as a whole new journey in the same project.

<Image align="center" src="https://files.readme.io/ebf96c3-pic44.png" />

### Note:

* Starting Node Configuration will be cleared while copying the journey. Bot designers have to reconfigure the Starting Node of the copied journey.
* Copied journeys need to be Deployed in order to make the changes live on the bot
