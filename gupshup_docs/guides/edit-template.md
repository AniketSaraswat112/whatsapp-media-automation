# Edit Template

> 📘
>
> * Templates can only be edited if the current status of template is **APPROVED**, **REJECTED** or **PAUSED**. \* **Template Category** can be edited only if the template is rejected due to Incorrect Category.

## Editing A Template

1. Go to Dashboard > WhatsApp > Your App > Templates, and click the **Arrow** button next to **Manage** on an existing template.

   <Image alt="Edit Template" align="center" border={true} src="https://files.readme.io/067707f-Screenshot_2024-05-29_at_4.09.07_PM.png">
     Edit Template
   </Image>
2. The original content of the template will be displayed.

   <Image alt="Original Content" align="center" border={true} src="https://files.readme.io/d068c8b-Screenshot_2024-05-29_at_9.56.32_PM.png">
     Original Content
   </Image>

   > 📘
   >
   > . The **Template Name**, **Language**, and **Template Labels** cannot be edited.. *Please note that edited templates will be submitted for approval again, regardless of the original status*.
3. Edit the required fields and select **Preview and Submit**.

   <Image alt="Edited Template" align="center" border={true} src="https://files.readme.io/a616cd0-Screenshot_2024-05-29_at_9.59.02_PM.png">
     Edited Template
   </Image>
4. Review the changes on the preview screen and select **Submit**.

   <Image alt="Preview of Edited Template" align="center" border={true} src="https://files.readme.io/a937f7c-Screenshot_2024-05-29_at_10.02.31_PM.png">
     Preview of Edited Template
   </Image>

The status will turn to **Pending**. You will be able to send this template again when it is approved.
