# Automation Workflows

1. To Create a Workflow rule, Go to **Settings** -> **Automation** -> **Workflow Rules**. Click **Create Rule** Button.

<Image alt="Workflow Rule " align="center" border={true} src="https://files.readme.io/1eba7c3-workflow_setting.png">
  Workflow Rule
</Image>

<Image alt="Create Rule" align="center" border={true} src="https://files.readme.io/cfb71e8-Create_rule.png">
  Create Rule
</Image>

2. Follow **Create New Rule** Popup window will open. Now,

* Choose **Module**
* Input a Rule **Name** & **Description**
* Click **Next**

<Image alt="Create New Rule" align="center" border={true} src="https://files.readme.io/d4dd4d1-Rule_name.png">
  Create New Rule
</Image>

<Image alt="Set Event " align="center" border={true} src="https://files.readme.io/69d416d-Set_event_for_workflow.png">
  Set Event
</Image>

3. Choose **Date/Time** or **event** to execute the rule
4. Select **condition**.

<Image alt="Select a Condition." align="center" border={true} src="https://files.readme.io/181e9c3-Set_condtion.png">
  Select a Condition.
</Image>

5. Select **Custom Action** As **WhasappMessageByGupshup**

<Image alt="Select Custom Action" align="center" border={true} src="https://files.readme.io/a18094e-Select_extension_in_work.png">
  Select Custom Action
</Image>

6. Select Channel

<Image alt="Select Channel" align="center" border={true} src="https://files.readme.io/6da1f6a-Select_channel_in_work.png">
  Select Channel
</Image>

7. Select **Receivers Number**

<Image alt="Select Receivers Number" align="center" border={true} src="https://files.readme.io/0f16ee4-Select_device_in_work.png">
  Select Receivers Number
</Image>

8. Select the category

<Image alt="Select Category " align="center" border={true} src="https://files.readme.io/198b24f-choose_category_in_work.png">
  Select Category
</Image>

9. Users can search for a template using the search bar or opt to filter templates by category, displaying only the selected templates in the list and Choose the desired Template from the List.

<Image alt="Select Template  " align="center" border={true} src="https://files.readme.io/43b2bbc-search_template_and_select_template_in_work.png">
  Select Template
</Image>

10. Then select variable if any from drop down.
11. Click on Save button to save workflow.

<Image alt="Save workflow " align="center" border={true} src="https://files.readme.io/fd7fd5f-save_workflow.png">
  Save workflow
</Image>

12. Enter Name and click on Save and Associate

<Image align="center" className="border" border={true} src="https://files.readme.io/7ba308a-saveandassosiate.png" />

13. Click **Save**. New Rule will show in Workflow Rules. Make sure status is ON.

<Image align="center" className="border" border={true} src="https://files.readme.io/edffee1-workflow_created.png" />
