# Set Callback URL

## Setting Callback URL

* Navigate to your **Dashboard** and click on the **Overview** button next to your App.

  <Image alt="overview button is next to App name(TestingDox) in this case" align="center" border={true} src="https://files.readme.io/c128207-Screenshot_2024-05-22_at_11.40.32_AM.png">
    Overview button is next to the App name(TestingDox) in this case
  </Image>
* Select the **Webhooks** tab and enter your Callback URL. Select **Set**.

  <Image alt="Enter Callback URL" align="center" border={true} src="https://files.readme.io/254bc77-Screenshot_2024-05-22_at_11.43.26_AM.png">
    Enter Callback URL
  </Image>
