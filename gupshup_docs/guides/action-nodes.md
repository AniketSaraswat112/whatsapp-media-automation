# Action Nodes

Various actions are required to be taken in designing an effective chatbot journey.\
Integrations in the journey, adding delay to the messages, waiting for an event, coding some use cases, and applying conditions to create branching in the flow are some actions covered in the Bot Studio.

Below are the Action Nodes available in Bot Studio. We will describe more about every node separately:
