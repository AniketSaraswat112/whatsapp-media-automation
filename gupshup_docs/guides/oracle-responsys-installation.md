# Installation

Installation Guide: Gupshup Oracle App for Oracle Responsys

Follow these steps to install the Gupshup Oracle App in your Oracle Responsys account. The installation process consists of three main steps: Installation, Granting Access, and Configuring Your Gupshup Console Account.

Installation Link:  [installation link](https://ams.oraclecloud.com/ams/applications/install/14218f24-7052-4167-a311-256b0fbadc59)

# Step 1: Log in to Your Oracle Responsys Account

1. **Log in to Oracle Responsys:**
   * Open your browser and navigate to the Oracle Responsys login page.
   * Enter your credentials and log in.
   * If prompted, verify the login by entering the verification code sent to your registered device.

# Step 2: Install the App

1. **Install the App Using the Provided Link:**
   * Open a new tab in your browser and click on the [installation link](https://ams.oraclecloud.com/ams/applications/install/14218f24-7052-4167-a311-256b0fbadc59) provided.
   * This will take you to the Oracle Responsys app installation page.
   * Once you click on the link, Oracle Responsys will redirect you to a page showing a list of all your available locations.
2. **Choose the Installation Location:**

   * From the list, select the **location** where you want to install the app.
   * After selecting the location, click on **Continue**.

   <Image alt="Choose Install location" align="center" border={true} src="https://files.readme.io/647242805a890d668b659556941f5ef027e299b1365fac7961bfb0455dd2a7bb-Screenshot_2024-12-02_at_3.21.07_PM.png">
     Choose Install location
   </Image>
3. **Fill in Required Fields (For Global Routed/OCI Accounts)**:

   * If you are using a global routed or OCI account, you will be prompted to fill in the following fields:
     * **Tenant ID** (optional)
     * **Site ID**
     * **Unique Identifier**\
       You can find the **Site ID** and **Unique Identifier** in your Oracle Responsys account login URL.
     * For example:https\://\<**UNIQUE\_ID**>.responsys.\<**SITE\_ID**>.com/authentication/login/LoginIdp
4. **Review and Accept the Terms and Conditions**:

   * After entering the required information, carefully read the app's **Terms and Conditions**.
   * Accept the terms to proceed.
5. **Install the App**:

   * After accepting the terms, the system will display a list of APIs used by the app.
   * Click the **Install** button to finalize the installation.

# Step 3: Confirm Installation Completion

1. **Check Installation Status**:

   * After the installation completes, navigate to **Oracle Responsys Account > Account Management > App Management**.
   * You should see the **Gupshup Oracle App** listed under Installed Apps.

   This confirms that the installation was successful.
